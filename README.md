## What JSX is
- JSX is JavaScript XML which allows us to write HTML code and describe UI components inside JavaScript code

## 3 rules of JSX
- single root element (<div></div> or <></>)
- close all tags (even those like img)
- use camelCase for Attributes to avoid using reserved words like "class"

## Example of embedded expression
- const name = "Jane"
- const element = <h1>Hello, {name}!</h1>
